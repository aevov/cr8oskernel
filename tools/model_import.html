Training Guide
How to prepare data for retraining

Data Format:

JSON array of training samples. Each sample can have:

  • input: Array of numbers (features)
  • target: Array of numbers (labels)

Example - XOR Training:

[
  {"input": [0, 0], "target": [0]},
  {"input": [0, 1], "target": [1]},
  {"input": [1, 0], "target": [1]},
  {"input": [1, 1], "target": [0]}
]

Tips:

  • More epochs = better learning (but slower)
  • Lower LR = more stable training
  • Include diverse training samples
